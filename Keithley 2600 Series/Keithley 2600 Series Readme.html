2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

·        VISA 4.0 or later

·        LabVIEW 8.0 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please send an email to applications@keithley.com.

This driver does not include support for contact check.

4. Revision History

The latest version of this instrument driver can be downloaded at the Keithley Instruments website.

 

REV 2.5.0, July-2012

Modified by: Keithley Engineering, Cleveland, OH

 

  • Added model support for 2601B, 2602B, 2604B, 2611B, 2612B, 2614B, 2634B, 2635B, and 2636B models.
  • Added model number check for TSP-Link VIs and Digital IO VIs because TSP-Link and Digital IO features are not supported by 2604B, 2614B and 2634B models.

 

REV 2.4.0, 11/03/2011

     Modified by: Keithley Engineering, Cleveland, OH

 

·        Added Over Voltage Protection support with Configure/Source/Source Overvoltage Protection VI for 2657A instruments.

 

REV 2.3.0, 03/07/2011

Modified by: Keithley Engineering, Cleveland, OH

 

  • Added Source Off Function support with Configure/Source/Source Off Function VI..
  • Added Source Off Limit Voltage support with Confingure/Source/Source Off Limit Voltage VI.
  • Added the Limit Current Mode Off function to the Sweep Config VI for 265xA instruments.

 

REV 2.2.0, 09/17/2010

 

     Modified by: Keithley Engineering, Cleveland, OH

 

  • Added Fast A/D support for 265xA instruments with the Configure/Measure Measure ADC Select VI.
  • Added Buffer Statistics support with Data/Reading Buffers Advanced Buffer Configuration and Buffer Statistics VIs
  • Added additional output control with Configure/Source Source Limit Power VIs.
  • Added support for asynchronous measurements in the Configure/Advanced Trigger Model Config Measure Event VI.

 

REV 2.1.3, 12/02/2009

Modified by: Keithley Applications, Cleveland, OH

·        Added support for Set and Query of High Capacitance Mode, in Configure/Source Source Hi-C Mode VI.

REV 2.1.2, 12/03/2008

Modified by: Keithley Applications, Cleveland, OH
